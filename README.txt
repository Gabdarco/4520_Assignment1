***Instructions on how to run the PrimeNumberCounter.java provided***

Step 1: Open the command prompt. To do this, press the windows key on your keyboard and type cmd, anc click on "Command Prompt".

Step 2: With the command prompt open, type "javac PrimeNumberCounter.java" and press Enter. You will see two .class files be created.
	You can ignore those.

Step 3: After the code compiles, you will then type "java PrimeNumberCounter" and press Enter. This should take around 15 seconds.

Step 4: After a few seconds the code will finish running, and a file named "results.txt" will appear in the same folder where 
	the PrimeNumberCounter.java file is located. This file will contain the results needed.
